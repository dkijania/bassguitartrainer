# bassguitartrainer

Bass Guitar Trainer is a free application for mastering a bass guitar. Application contains also metronome and drum machine

Bass Guitar Trainer is a free application for mastering a bass guitar. This application is a easy way to learn music notation, notes positions on the fretboard. Application contains also metronome with standard or custom time signatures and a drum machine which can fake real percussion with full set of drums and hi-hats.
